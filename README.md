# このリポジトリについて
## これは何？
日本政府による 「政府標準利用規約 (第1.0版) 」 (仮称)  (案) をMarkdownとして書きなおしたものです。
http://www.kantei.go.jp/jp/singi/it2/densi/dai6/siryou2.pdf

## 実施して大丈夫なの？
